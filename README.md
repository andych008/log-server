# log-server

show remote log. work with [log-client](https://github.com/andych008/log-client)

logcat --> log-server --> log-client 

## 开始
```
//install typescript globally
npm i -g typescript

npm i

// please compile my code
tsc

// then run the server
node ./dist/server/server.js
```
